# The Margins — post footer (reusable sign-off)

Substack has no global "footer" setting and no custom HTML/CSS — it auto-appends
the required legal + unsubscribe footer to every email. The footer you *control*
is the sign-off block at the bottom of each post. Paste one of the versions
below at the end of a post (Substack converts the markdown links on paste).

**Tips**
- For the actual **Subscribe button**, use Substack's native button: in the
  editor type `/button` → "Subscribe", instead of (or above) the text link.
- The `♡` renders as plain text — on brand, and works everywhere.
- The 30 drafts (`05`–`34`) already have their own tailored CTAs, so only add
  this to *new* posts, or swap it in if you prefer one consistent footer.
